Unstructured Data & NoSQL Technologies

This course introduces modern database technologies used to handle massive volumes of structured, semi-structured and unstructured data. Students learn NoSQL concepts, MongoDB, document databases, key-value databases, graph databases, distributed systems and big data technologies. The course focuses on real-world applications used in cloud computing, IoT systems, social media platforms and enterprise-scale data processing.
